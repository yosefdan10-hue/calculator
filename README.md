מחשבון פשוט שנבנה במהלך לימוד Git.

## קבצים בפרויקט

- main.py - הקובץ הראשי
- operations.py - פעולות בסיסיות (חיבור, חיסור, כפל, חילוק)
- advanced.py - פעולות מתקדמות (חזקה, שורש)
- ui.py - ממשק משתמש
- stats.py - פונקציות סטטיסטיות: ממוצע, מקסימום, מינימום, חציון

## איך מריצים

```bash
python main.py
```
